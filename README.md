⌨️ Typing Speed Test
A typing speed test web app that measures your WPM (Words Per Minute) and accuracy in real time — with difficulty levels and a neon dark theme UI.
🔗 Live Demo

View Live Project
✨ Features

Real-time letter-by-letter highlighting (green = correct, red = wrong)
Live WPM and accuracy calculation as you type
60 second countdown timer
Easy, Medium, and Hard difficulty levels
Random paragraph selection every round
High score saved using LocalStorage
Neon dark theme UI
Fully responsive design

🛠️ Built With

HTML5
CSS3
JavaScript (Vanilla JS)

🚀 How to Run Locally

Clone this repository: git clone https://github.com/f250594-stack/Typing-speed-test.git
Open index.html in your browser or use Live Server in VS Code
